# Plain White Shop

Este es un proyecto para una entrevista técnica (LOGO INTERVIEW).

## Descripción

Una página web sencilla construida con:
- HTML5
- Tailwind CSS (vía CDN)
- Google Fonts (Science Gothic, Roboto, Quicksand)

## Estructura

- `index.html`: Archivo principal con la estructura y estilos.
- `assets/`: Carpeta que contiene recursos como imágenes (logo).

## Cómo ver el proyecto

Simplemente abre el archivo `index.html` en cualquier navegador web.
